# NovaLike

NovaLike est l’IA communautaire de l’écosystème WanaLike.

## Fonctionnalités MVP

- accueil des nouveaux membres
- message automatique de bienvenue
- présence Discord personnalisée
- commandes de base
- architecture prête pour l’IA

## Stack

- Python
- discord.py
- python-dotenv

## Installation

```bash
git clone https://github.com/KoS-aka-NiX/novalike.git
cd novalike

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

## Configuration

```bash
cp .env.example .env
nano .env
```

Ajouter le token Discord dans `.env`

## Lancement

```bash
python3 bot/main.py
```

## Commandes

```txt
!ping
!nova
```

## Vision

Créer une véritable IA communautaire cross-platform capable d’interagir naturellement avec les membres tout en renforçant l’activité et l’identité de l’écosystème WanaLike.
